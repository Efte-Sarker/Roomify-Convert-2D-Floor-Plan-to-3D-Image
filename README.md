<h1>Roomify</h1>

<p>Roomify is a full-stack web application designed to bridge the gap between architectural
imagination and visual reality. The platform enables users to design residential floor plans from
scratch using an interactive 2D editor, and then users can preview their created plan in a 3D
model. Also, it can generate photorealistic top-down 3D renders of those designs using artificial
intelligence. Alternatively, users can upload an existing 2D floor plan image and instantly
receive an AI-rendered 3D visualization of the space.
<br>The application addresses a genuine pain point in early-stage architectural and interior planning:
the high cost and technical barrier of 3D rendering software. By combining a purpose-built
browser-based floor plan editor with Google's Gemini multimodal AI model, Roomify
democratizes 3D space visualization for homeowners, architecture students, and design
enthusiasts who would otherwise rely on expensive professional tools.
<br>At its core, Roomify is a single-page application (SPA) with a React frontend and an Express
backend that manages users, projects, and AI rendering requests. All user work is persistently
saved to a MySQL database and is accessible via a structured History page, making Roomify a
complete design-to-render workflow tool.</p>
